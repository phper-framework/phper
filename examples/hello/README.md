# hello

Hello world example.

## Environment

```bash
# Optional, specify if php isn't installed globally.
export PHP_CONFIG=<Your path of php-config>
```

## Build

```bash
cargo build --release
```

## Run

```bash
php -d "extension=target/release/libhello.so" -r "say_hello('Bob');"
```
